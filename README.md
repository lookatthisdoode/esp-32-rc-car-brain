# Arduino based brains for a lego technic RC car.

I am building lego technic RC car and since i have some arduinos laying around I decided why not make it DS4 controlled as well.

# Functions
- Oled display with rotary encoder to change car settings on the fly
- Manual servo and motor control from the menu
- Dualshock 4 control

# Deps list
Most libraries bundled, but make sure to install [Bluepad32](https://bluepad32.readthedocs.io/en/latest/plat_arduino/) board.
